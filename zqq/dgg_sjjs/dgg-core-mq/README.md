### 架构中心-消息队列工程
##### 说明：

该工程主要负责消息队列,所有架构中心的项目都必须在pom.xml如下代码：
```
<parent>
    <groupId>net.dgg.core.parent</groupId>
    <artifactId>dgg-core-parent</artifactId>
    <version>RELEASE</version>
</parent>
```

##### Maven引入方式：
```
<dependency>
	<groupId>net.dgg.core.mq</groupId>
    <artifactId>dgg-core-mq</artifactId>
</dependency>
```
